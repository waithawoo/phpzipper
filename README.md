 - **[Installation](#installation)**
 - **[Usage](#usage)**

## Installation

Install this package via [Composer](https://getcomposer.org/).

```
composer require waithaw/phpzipper
```

## Usage

### Create Zip object

```php
use WaiThaw\PhpZipper\Zip;

$zip = new Zip();
```
